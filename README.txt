WEBSHOP APP
-----------------------------------

1. Run docker file
2. In case it will be necessary run script of database file
3. Import Project with MAVEN or INTELLIJ IDEA. 
   	run liquidabase pluggin or file with command.
	run "clean install" if you dont need execute test 
	run "clean install -DskipTests". 
	run "spring-boot:run". configure in port:8080.

4. Install Postman App and import "webshop-back.postman_collection.json" in postman file
5. Can review last integration testing in test file
